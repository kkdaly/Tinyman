# 知识库索引

Agent 回答用户问题时：先读本文件定位文档 → 再读对应文档 → 必要时读 `repos/` 代码确认。

## 项目文档

| 文档 | 内容 | 什么时候看 |
|------|------|-----------|
| `your-project.md` | 项目概述、技术栈、踩坑记录、常见问题 | 所有问题首选 |
| _(按需添加更多)_ | | |

## 添加新文档

1. 在 `knowledge-base/` 下新建 `xxx.md`
2. 在上方表格加一行，写明文档内容和什么时候看
3. Agent 下次回答时自动能查到

Agent 无法回答的问题 → 记录下来 → 补充到对应文档 → Agent 下次就会了。
